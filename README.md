# 🕒 WatchStore – E-Commerce Website

A modern and responsive **watch e-commerce website** built using **HTML, CSS, and JavaScript**. This project showcases a functional online store interface with features like product browsing, cart management, wishlist, and checkout flow.

---

## 🚀 Features

* 🏠 **Home Page**

  * Hero banner with sliding images
  * New arrivals and product listings

* 🛍️ **Product Pages**

  * Dedicated pages for individual watches (p1–p5)
  * Product details and visuals

* 🛒 **Cart System**

  * Add/remove items from cart
  * Dynamic cart count update

* ❤️ **Wishlist**

  * Save products for later
  * Separate wishlist page

* 💳 **Checkout Page**

  * Simulated checkout flow

* ℹ️ **About Us Page**

  * Information about the store

---

## 🛠️ Tech Stack

* **Frontend:**

  * HTML5
  * CSS3
  * JavaScript (Vanilla)

* **Assets:**

  * Images stored in `/assets` folder
